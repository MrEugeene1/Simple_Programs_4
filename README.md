# Python Works

A beginner-friendly Python practice project demonstrating a linear search algorithm.

## Overview

The program scans a predefined list of integers to find a target value and prints whether the value exists, including its index when found.

## File Structure

- `simplePrograms4.py` - Linear search implementation.

## Run

```bash
python simplePrograms4.py
```

## Example Output

```text
Element found at index 4
```
